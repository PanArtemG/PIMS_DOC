Build and migrate

vapor build && vapor run migrate

Run OpenAPI

1. Run project.
2. Go to URL: 
    https://taskerswift-swagger.azurewebsites.net/#/
3. Embed URL: 
    http://127.0.0.1:8084/v1/open-api

Work with Docker

Run PostgreSQL command

docker run --name person_service -e POSTGRES_DB=person_service -e POSTGRES_USER=person_service -e POSTGRES_PASSWORD=password -p 5434:5432 -d postgres

Run PostgreSQL command for test DB

docker run --name person_service-test -e POSTGRES_DB=person_service-test -e POSTGRES_USER=person_service-test -e POSTGRES_PASSWORD=password -p 5435:5432 -d postgres

Stop DB command

docker stop postgres

Delete DB command

docker rm postgres

Run MongoDB command

docker run --name person_catalogs -p 27014:27017 -d mongo

Run MongoDB command for test DB

docker run --name person_catalogs-test -p 27015:27017 -d mongo

Stop MongoDB command

docker stop mongo

Delete MongoDB command

docker rm mongo
